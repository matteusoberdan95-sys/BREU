# Guia de assets reutilizaveis

Para objetos repetidos, criar 2 a 4 variacoes base.

Exemplo:

```text
rock_A
rock_B
rock_C
```

Depois duplicar:

```text
rock_A_01
rock_A_02
rock_B_01
```

## Regras

- Usar `Shift+D` para copia independente.
- Usar `Alt+D` para instancia vinculada quando fizer sentido.
- Variar escala, rotacao e posicao.
- Nao modelar tudo do zero.
- Criar biblioteca `LIB_Reusable` no Blender.
- Objetos como pedras, tabuas, cactos, cruzes, potes, caixas e galhos devem seguir esse padrao.

## Quando usar duplicacao

| Tipo | Variacoes base |
|------|----------------|
| Pedras | 3 variacoes |
| Cerca | 1 poste + 1 trilho + 1 trilho quebrado |
| Cactos | 2 variacoes |
| Cruzes | 2 variacoes |
| Tabuas | 3 variacoes |
| Potes | 2 variacoes |
| Ossos | 3 variacoes |

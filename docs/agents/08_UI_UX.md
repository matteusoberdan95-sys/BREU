# Agente UI / UX

## Objetivo

Dar feedback claro com HUD minimo, sem quebrar a imersao.

## Responsabilidades

- Prompt de interacao.
- Indicador de stamina.
- Indicador de bateria.
- Arma equipada e durabilidade.
- Leitura de documentos.
- Menus futuros.

## Entregas esperadas

- HUD legivel.
- UI de bilhete.
- Feedback de estado critico.

## Limites

- Nao poluir a tela.
- Nao explicar mecanicas com texto excessivo dentro do jogo.

# Agente Art Pipeline / Blender

## Objetivo

Preparar assets do Blender para entrar no Godot sem quebrar escala, colisao ou organizacao.

## Responsabilidades

- Escala e nomenclatura.
- Exportacao `.glb`.
- Pivos e transforms.
- Colisoes simples.
- Materiais e modularidade.

## Entregas esperadas

- Guia de exportacao.
- Lista de assets por prioridade.
- Validacao visual no Godot.

## Limites

- Nao cria gameplay em assets.
- Nao entrega modelos finais sem passar por teste de escala.

## Proximos passos

- Criar cama de ferro placeholder final.
- Criar martelo rustico.
- Criar kit de paredes mofadas.

## Checklist

- [ ] Transform aplicado.
- [ ] Nome limpo.
- [ ] Pivo correto.
- [ ] Exportado como `.glb`.
- [ ] Colisao simples definida.

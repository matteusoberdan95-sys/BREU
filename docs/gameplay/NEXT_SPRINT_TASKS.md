# BREU - Proximas tarefas

## Como usar este arquivo

Este e o backlog vivo do projeto. Ao abrir o projeto no Codex, Cursor IDE ou Cursor CLI, leia primeiro `docs/PROJECT_STATE.md` e `docs/HANDOFF.md`, depois use este arquivo para escolher a proxima tarefa.

Ao terminar uma tarefa, marque o estado real em `docs/PROJECT_STATE.md` e registre o handoff em `docs/HANDOFF.md`.

# Direção de Arte dos Cenários

## Estilo

Realismo estilizado sujo, feito para Godot, com assets criados no Blender.

Não buscar hiper-realismo inicialmente. Priorizar:

- Silhueta forte
- Composição clara
- Materiais sujos
- Iluminação dramática
- Objetos narrativos
- Performance

## Cenário base

Interior do Nordeste, anos 70/80.

## Paleta

| Cor | Uso |
|-----|-----|
| Barro seco | Paredes, chão externo |
| Bege encardido | Reboco, tecido |
| Marrom madeira velha | Portas, móveis, cruzes |
| Cinza cimento | Pisos, pedra |
| Preto de sombra | Corredores, cantos |
| Amarelo de vela | Iluminação quente |
| Vermelho escuro | Sangue seco |
| Verde mofo | Umidade, decomposição |
| Ferrugem | Ferramentas, grades |

## Pipeline

1. Blockout no Blender
2. Export `.glb`
3. Teste no Godot
4. Ajuste escala/colisão
5. Reexport
6. Polimento com material, luz, áudio e props

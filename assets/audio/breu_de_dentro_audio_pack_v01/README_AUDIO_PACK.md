# BREU DE DENTRO — Audio Pack v0.1

Pacote inicial de áudio `.ogg` para o Quarto 07 e corredor.

Copie a pasta `assets/audio/` para a raiz do projeto Godot:

```text
res://assets/audio/
```

## Observação
Esses sons são originais e gerados proceduralmente. Servem como base realista/atmosférica para playtest.
Para a versão final, o ideal é substituir ou misturar com Foley gravado.

## Volumes sugeridos
- Ambiência: -20 dB a -14 dB
- Rádio: -18 dB a -10 dB
- Porta: -8 dB a -4 dB
- Susto: -4 dB a 0 dB
- Inimigo: -12 dB a -6 dB
